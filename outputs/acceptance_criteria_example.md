# Example Acceptance Criteria (Generated)

- Given a logged-in user is on the profile page, when they update valid name, email, phone and address and click Save, then the system should save the changes and show a success message.

- Given a user enters an email that already exists for another account, when they click Save, then the system should show an error message and not save the changes.

- Given a user enters a phone number without a valid country code, when they click Save, then the system should show a validation error.

- Given the profile page is requested by a logged-in user, then the page should load within 3 seconds under normal network conditions.
